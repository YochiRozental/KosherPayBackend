# KosherPay Backend 

This is the backend service for **KosherPay**, a modern payment management system. 
Built with **FastAPI** for high performance and **PostgreSQL** for reliable data management.

## Features
* **Modular Architecture**: Clean separation of concerns (Routes, Services, Repositories).
* **Authentication**: Secure user login and registration using JWT and Bcrypt.
* **Database Management**: Integrated with SQLAlchemy and Pydantic for data validation.
* **Security**: Environment variables management for sensitive data.

## Tech Stack
* **Framework**: [FastAPI](https://fastapi.tiangolo.com/)
* **Database**: PostgreSQL with SQLAlchemy ORM
* **Validation**: Pydantic v2
* **Security**: PyJWT & Bcrypt
